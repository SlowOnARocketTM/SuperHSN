# HSN+

HSN+ is a clean match browser that reads the Streamed PK API, filters to football and Formula 1 events, and shows the available external stream embeds in one focused interface.

## What it does

- Fetches sports and match data from the Streamed PK API
- Keeps only football and Formula 1 content
- Sorts the feed into a clean match list with posters and team badges
- Loads an external embed for the selected match
- Shows a first-visit warning that the site does not host content

## Run locally

1. Install dependencies with `npm install`.
2. Start the dev server with `npm run dev`.
3. Open the local URL shown in the terminal.

## Notes

The site uses external API data and embeds only. It does not host any video or media itself.
